# patterns_LAB1
Patterns lab 1

Демонстрация паттернов проектирования GoF в C++
Эта консольная программа наглядно демонстрирует одновременное применение четырёх классических паттернов проектирования:

Prototype (Прототип)
Singleton (Одиночка)
Abstract Factory (Абстрактная фабрика)
Builder (Строитель)

в рамках одной небольшой, но полностью рабочей системы - графической сцены с примитивами: точками, линиями и окружностями.
Программа написана на C++11 с использованием Qt только для организации консольного приложения (QCoreApplication). Всё остальное - чистый C++, без графического интерфейса.

Особенности реализации:
«Хук» в фабриках - при создании любого объекта через ColorGraphFactory / BWGraphFactory он автоматически добавляется в синглтон-сцену.
Два разных Builder’а на одной и той же последовательности шагов:
RealSceneBuilder - создаёт реальные объекты и рисует сцену.
MemorySceneBuilder - только подсчитывает объём памяти (полезно для оценки производительности).

Директор (SceneDirector) полностью отделён от конкретных строителей и фабрик - легко добавлять новые виды сцен.

Запуск и вывод
Программа выводит в консоль три демонстрационных блока:
text=== Colored scene (Abstract Factory + Real Builder + Singleton + Hook) ===
=== What the scene contains ===
Color Point (10, 20)
Color Line (0, 0) - (100, 50)
Color Circle (50, 50) r = 25

=== Memory count (Memory Builder) ===
Total memory to use is: XXX bytes

=== Prototype demonstration ===
Color Point (5, 5)
